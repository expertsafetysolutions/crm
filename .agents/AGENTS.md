# Custom Agent Rules

- **Feature Preservation**: Always preserve existing features, badges, icons, warning indicators, and styling unless explicitly requested to be removed by the user. If any proposed change could impact, hide, or remove an existing feature, always discuss it and obtain user confirmation first. Modifying or editing features to make them better or smaller is encouraged, but they must not be lost or broken.

- **Vercel Deployments**: Never run `npx vercel` or push directly to Vercel deployment environments automatically. Only deploy when explicitly asked or instructed by the user.
